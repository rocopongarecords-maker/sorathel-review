# Continuity Checking Skill

## When to Use
- Before writing a new chapter (quick check on relevant facts)
- After completing an episode (full check across all chapters)
- When user asks for a continuity audit

## Checklist
1. **Terminology:** "Amarvel" for roots, "Amarwen" for the Tree. Ethura NEVER in character voice.
2. **Physical descriptions:** Do characters look the same as previously described?
3. **Timeline:** Time of day, travel distances, sequence of events consistent?
4. **Character knowledge:** Does anyone know something they shouldn't know yet?
5. **Relationships:** Are dynamics consistent with what's been shown?
6. **World details:** Food, architecture, creatures, geography match previous chapters?
7. **Magic mechanics:** Does the Amarvel work the same way as previously shown?
8. **Emotional continuity:** Would this character react this way given what they've been through?

## How to Report
For each issue found:
```
[CONTRADICTION] Ch## vs Ch##: [description of mismatch]
[MINOR] Ch##: [small inconsistency, easily fixed]
[QUESTION] Ch##: [ambiguity that should be resolved]
```

## Cross-Reference Files
- `bible/story-bible.md` — primary source of truth
- `bible/terminology.md` — term definitions
- `characters/*.md` — character details as shown
- Previous chapters in `episodes/ep##/`
