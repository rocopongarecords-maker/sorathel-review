# Story Bible Update Skill

## When to Use
After every chapter is written or revised, update the Story Bible with new canon.

## Process
1. Read the chapter just written/revised
2. Identify NEW information revealed on page:
   - Character physical descriptions not previously recorded
   - Character behaviors/habits shown for the first time
   - Relationship dynamics shown or changed
   - World details (places, food, creatures, architecture)
   - Magic/network mechanics demonstrated
   - Timeline/sequence information
3. Add each new detail to the appropriate Story Bible section with chapter citation
4. If a Writing Rule was violated and caught, add the rule to §7
5. Update the terminology table if new terms appeared
6. Check for contradictions between the new chapter and existing Bible entries

## Format for New Entries
```
**[Detail]:** [Ch #] Description as shown in prose. Quote the specific language if it matters.
```

## What NOT to Add
- Author-level truth that wasn't shown on page (that stays in the Anomaly Bible)
- Reader interpretations (only what the text actually says)
- Planned future details (those stay in outlines)
